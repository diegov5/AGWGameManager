Game Day Instructions It’s Game Day - time to setup the game, get into costume and assemble your participants! Game Day and Time

You’ve indicated that you’ll be playing your game on December 14th at 10:00AM.

Based on your game date and time, we’ve created this handy instruction list to help you get ready for the game and set it up on game day.



8:00AM - Setup the Hospital

The Hospital will take some time to set up so let’s get it ready early. Bring all of the safety suit equipment and place it so the Nurse can easily distribute the equipment to the players when they visit the Hospital after the kickoff.

Place the Zombie makeup and shirts in a bag or box so that they are available in the Hospital but not visible to the players.

Hang your Hospital Sign on the door of your Hospital location.

If you haven’t done already, make sure the Puzzle 1 Safety Suit Instructions have been installed in the pockets of the Safety Suits.



8:10AM - Setup the Armory

The Armory is a key location near the end of the game. You’ll want to leave the nerf guns and bullets you wish to hand out to the surviving humans when they visit the Armory.

Hang your Armory Sign on the door of your Armory location.



8:20AM - Setup Puzzles 2 Thru 6

You’ll need 20-30 minutes to set up Puzzles 2 thru 6. Before you leave to install the puzzles, make sure you have all of the puzzles with you. You can choose to set up each puzzle in the game order or by location - whichever is most convenient for you.



8:50AM - Setup the Signal Flags

Next, place the five signal flags in the Puzzle 7 location. You can control the difficulty of this puzzle by choosing the distance and visibility of the signal flags.



9:00AM - Setup the Extraction Point

Place the Signal Flare at the location of the Extraction Point and end of the game. Turn the Signal Flare light on.



9:10AM - Game Host and Nurse Costumes

The game is now set up. If you haven’t already, it’s time for you and the Nurse to put on their costumes.



9:30AM - Review

It’s now thirty minutes before the game will begin. This is a good time to review that everything is in place and ready to go.



10:00AM - Game On!

Proceed to the Kickoff Location and greet the game’s participants. When everyone is assembled and ready to play, it’s time for you to give your Bad News Speech to start the game.
